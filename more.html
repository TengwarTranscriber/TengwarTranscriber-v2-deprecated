To install a TTF Font:

Windows:

  1. Download it
  2. Right-click the downloaded font file
  3. Select “Install”

Mac:

  1. Download it
  2. Double-click the downloaded font file to preview it
  3. Click Install Font in the preview window to install it

Linux:

Different Linux distributions come with different desktop environments, and those different desktop environments contain different applications for this.

  1. Download it
  2. Double-click the font to preview it
  3. On Ubuntu or other GNOME-based distributions, GNOME Font Viewer will open
  4. Click the Install button to install the font for your user account
